## ما هو EchoForge؟

**EchoForge** مغيّر صوت يعمل بالكامل دون اتصال لـ Android وiPhone. سجّل صوتك، وطبّق أحد 24 تأثيرًا مضبوطًا بعناية (أو ابنِ سلسلة خاصة بك)، وشارك النتيجة مباشرة في تطبيقات المراسلة. كل شيء — التسجيل، المعالجة، تشفير الملف الصوتي — يحدث على الجهاز. لا إعلانات، لا حساب، لا إذن إنترنت، لا تتبع.

ما يميّزه: **سلاسل تأثيرات قابلة للتكديس**. يمكنك تجميع pitch وreverb وecho وdistortion وbitcrush والمزيد عبر السحب والإفلات وحفظها كصوتك المخصص. معظم التطبيقات تخفي هذا المستوى خلف اشتراك. EchoForge يبقيه مجانيًا.

## أهم الميزات

### 24 إعدادًا مسبقًا
- **هيليوم**، **سنجاب**، **روبوت**، **شيطان**، **شبح**، **وحش**، **فضائي**
- **صوت عميق**، **عملاق**، **كهف**، **استاد**، **صدى**
- **Lo-Fi**، **فينيل**، **همس**
- **حمام**، **Bass Boost**، **Flanger**، **مكبّر**، **هاتف**، **آركيد**، **DJ**
- خانات **Custom** لإبداعاتك

### سلاسل تأثيرات مخصصة
- **كدّس** pitch، reverb، echo، distortion، bitcrush، EQ
- **رتّب** بالسحب والإفلات
- **اضبط** كل معلمة بدقة
- **تراجع / تكرار**
- **احفظ** السلسلة كإعداد شخصي

### المشاركة
- **WhatsApp**، **Telegram**، **Discord**، **Messenger**، **البريد**، **الملفات**

EchoForge يصدّر ملفات صوت حقيقية بصيغة وبيانات وصفية صحيحة.

### الخصوصية
- **100% دون اتصال**
- **بدون إعلانات**
- **بدون تتبع**
- **بدون حساب**
- **الصوت يبقى على الجهاز**

## حالات الاستخدام

| الحالة | ماذا يفعل EchoForge |
|--------|----------------------|
| رسالة WhatsApp مرحة | سجّل → سنجاب → شارك |
| رسالة هالوين | سجّل → شيطان + كهف → شارك |
| ملاحظة Discord | سجّل → روبوت + Bitcrush → Discord |
| مقطع ASMR | سجّل → همس → احفظ |
| مزحة Telegram | سجّل → هيليوم → ملاحظة صوتية |
| اختبار كتاب صوتي | سجّل → صوت عميق → احفظ |
| تعليق صوتي TikTok | سجّل → سلسلة مخصصة → احفظ |
| مقدمة بودكاست | سجّل → مكبّر / DJ → احفظ |
| مشهد "عبر الهاتف" | سجّل → هاتف → شارك |
| صوت شخصية | سجّل → وحش + Distortion → احفظ |

## الأسئلة الشائعة

**هل هو مجاني فعلًا؟**
نعم.

**هل أحتاج حسابًا؟**
لا.

**هل يستخدم الإنترنت؟**
لا.

**هل يجمع البيانات؟**
لا.

**لماذا يحتاج إلى الميكروفون؟**
لتسجيل الصوت.

**لماذا يحتاج إلى الملفات/الوسائط؟**
لحفظ الملف الصوتي ومشاركته.

**ما الصيغ التي يصدّرها؟**
صيغ قياسية متوافقة مع WhatsApp وTelegram وDiscord وغيرها.

**أقصى مدة تسجيل؟**
لا يوجد حد ثابت.

**تغيير الصوت لحظيًا أثناء المكالمة؟**
لا. EchoForge يعالج المقاطع المسجلة.

**ما الأجهزة المدعومة؟**
Android بميكروفون، iPhone / iPad بنظام iOS 13.0 أو أحدث.

**كيف أبلغ عن خطأ؟**
issue على [github.com/Lapnito/echoforge/issues](https://github.com/Lapnito/echoforge/issues) أو tom@lapnito.cz.
