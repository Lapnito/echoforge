## EchoForge란?

**EchoForge**는 Android와 iPhone을 위한 완전 오프라인 보이스 체인저입니다. 목소리를 녹음하고, 정성껏 다듬어진 24가지 효과 중 하나(또는 직접 만든 효과 체인)를 적용해, 그 결과를 메신저에 바로 공유합니다. 녹음·처리·오디오 파일 인코딩까지 모두 기기 내에서 이뤄집니다. 광고 없음, 계정 없음, 인터넷 권한 없음, 추적 없음.

특별한 점: **쌓을 수 있는 효과 체인**. 피치·리버브·에코·디스토션·비트크러시 등을 드래그 앤 드롭으로 조합하고, 나만의 목소리로 저장합니다. 대부분의 앱이 구독 뒤로 숨기는 기능을 EchoForge는 무료로 둡니다.

## 주요 기능

### 24가지 보이스 프리셋
- **헬륨**, **다람쥐**, **로봇**, **악마**, **유령**, **괴물**, **외계인**
- **저음**, **거인**, **동굴**, **스타디움**, **에코**
- **Lo-Fi**, **바이닐**, **속삭임**
- **욕실**, **Bass Boost**, **Flanger**, **메가폰**, **전화**, **아케이드**, **DJ**
- 자작용 **Custom** 슬롯

### 커스텀 체인
- **쌓기** — 피치, 리버브, 에코, 디스토션, 비트크러시, EQ
- 드래그로 **순서 변경**
- 슬라이더 **미세 조정**
- **실행 취소 / 다시 실행**
- **저장**해 개인 프리셋으로

### 공유
- **WhatsApp**, **Telegram**, **Discord**, **Messenger**, **이메일**, **파일**

EchoForge는 형식과 메타데이터가 올바른 진짜 오디오 파일을 내보냅니다.

### 개인정보 보호
- **100% 오프라인**
- **광고 없음**
- **추적 없음**
- **계정 없음**
- **오디오는 기기에 남음**

## 사용 시나리오

| 상황 | EchoForge가 하는 일 |
|------|----------------------|
| WhatsApp 재미 음성 | 녹음 → 다람쥐 → 공유 |
| 핼러윈 메시지 | 녹음 → 악마 + 동굴 → 공유 |
| Discord 메모 | 녹음 → 로봇 + Bitcrush → Discord |
| ASMR 클립 | 녹음 → 속삭임 → 저장 |
| Telegram 장난 | 녹음 → 헬륨 → 음성 메시지 |
| 오디오북 테스트 | 녹음 → 저음 → 저장 |
| TikTok 보이스오버 | 녹음 → 커스텀 체인 → 저장 |
| 팟캐스트 인트로 | 녹음 → 메가폰 / DJ → 저장 |
| "전화 너머" 연출 | 녹음 → 전화 → 공유 |
| 캐릭터 보이스 | 녹음 → 괴물 + Distortion → 저장 |

## 작동 방식

**오디오 처리는 어디서 이뤄지나요?**
완전히 기기 내에서. 어떤 단계에서도 인터넷이 필요하지 않습니다.

**녹음이 업로드되나요?**
절대 아닙니다.

**효과가 정말 쌓이나요?**
네.

**공유가 안정적인 이유는?**
EchoForge가 형식·샘플레이트·메타데이터가 올바른 진짜 오디오 파일을 내보내기 때문입니다.
