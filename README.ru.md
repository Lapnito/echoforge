## Что такое EchoForge?

**EchoForge** — полностью офлайн изменитель голоса для Android и iPhone. Запишите свой голос, примените один из 24 тщательно настроенных эффектов (или соберите собственную цепочку) и поделитесь результатом прямо в мессенджерах. Запись, обработка и кодирование аудиофайла — всё на устройстве. Без рекламы, без аккаунта, без разрешения на Интернет, без слежки.

Уникальность: **стекаемые цепочки эффектов**. Pitch, ревербератор, эхо, дисторшн, битхрашер и многое другое можно объединять, перетаскивать и сохранять как собственный голос. В большинстве приложений это спрятано за подпиской. EchoForge оставляет бесплатно.

## Главные возможности

### 24 пресета голоса
- **Гелий**, **Бурундук**, **Робот**, **Демон**, **Призрак**, **Монстр**, **Инопланетянин**
- **Глубокий голос**, **Великан**, **Пещера**, **Стадион**, **Эхо**
- **Lo-Fi**, **Винил**, **Шёпот**
- **Ванная**, **Bass Boost**, **Flanger**, **Мегафон**, **Телефон**, **Аркада**, **DJ**
- **Custom** для собственных вариантов

### Цепочки эффектов
- **Стек** — pitch, ревербератор, эхо, дисторшн, битхрашер, EQ
- **Сортировка** перетаскиванием
- **Тонкая настройка** ползунками
- **Отмена / повтор**
- **Сохранение** цепочки как пресета

### Обмен
- **WhatsApp**, **Telegram**, **Discord**, **Messenger**, **E-mail**, **Файлы**

EchoForge экспортирует настоящие аудиофайлы с корректным форматом и метаданными.

### Конфиденциальность
- **100% офлайн**
- **Без рекламы**
- **Без слежки**
- **Без аккаунта**
- **Аудио остаётся на устройстве**
