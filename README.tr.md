## EchoForge nedir?

**EchoForge**, Android ve iPhone için tamamen çevrimdışı çalışan bir ses değiştiricidir. Sesinizi kaydedin, dikkatle ayarlanmış 24 efektten birini uygulayın (ya da kendi efekt zincirinizi oluşturun) ve sonucu doğrudan mesajlaşma uygulamalarına gönderin. Kayıt, işleme ve dosya kodlama dahil her şey cihazda olur. Reklam yok, hesap yok, internet izni yok, takip yok.

Ayırt edici özellik: **istiflenebilir efekt zincirleri**. Pitch, reverb, echo, distortion, bitcrush ve daha fazlasını sürükle-bırak ile birleştirip kendi sesinizi olarak kaydedebilirsiniz. Çoğu uygulama bu seviyeyi abonelik arkasına saklar. EchoForge ücretsiz tutar.

## Temel özellikler

### 24 ses preset'i
- **Helyum**, **Sincap**, **Robot**, **Şeytan**, **Hayalet**, **Canavar**, **Uzaylı**
- **Derin ses**, **Dev**, **Mağara**, **Stadyum**, **Echo**
- **Lo-Fi**, **Plak**, **Fısıltı**
- **Banyo**, **Bass Boost**, **Flanger**, **Megafon**, **Telefon**, **Arcade**, **DJ**
- Kendi yaratımlarınız için **Custom** slotlar

### Özel efekt zincirleri
- **İstifle** pitch, reverb, echo, distortion, bitcrush, EQ
- **Sırala** sürükle-bırak ile
- **İnce ayar** sürgülerle
- **Geri al / yinele**
- Kendi preset'iniz olarak **kaydet**

### Paylaşım
- **WhatsApp**, **Telegram**, **Discord**, **Messenger**, **E-posta**, **Dosyalar**

EchoForge gerçek ses dosyalarını doğru format ve meta verilerle dışa aktarır.

### Gizlilik
- **%100 çevrimdışı**
- **Reklamsız**
- **Takipsiz**
- **Hesapsız**
- **Ses cihazda kalır**

## Kullanım senaryoları

| Senaryo | EchoForge ne yapar |
|---------|---------------------|
| WhatsApp komik sesli mesaj | Kaydet → Sincap → paylaş |
| Cadılar Bayramı mesajı | Kaydet → Şeytan + Mağara → paylaş |
| Discord notu | Kaydet → Robot + Bitcrush → Discord |
| ASMR klip | Kaydet → Fısıltı → kaydet |
| Telegram şakası | Kaydet → Helyum → sesli mesaj |
| Sesli kitap denemesi | Kaydet → Derin ses → kaydet |
| TikTok seslendirme | Kaydet → özel zincir → kaydet |
| Podcast jeneriği | Kaydet → Megafon / DJ → kaydet |
| "Telefonda" drama | Kaydet → Telefon → paylaş |
| Karakter sesi | Kaydet → Canavar + Distortion → kaydet |
