## Що таке EchoForge?

**EchoForge** — повністю офлайн зміна голосу для Android та iPhone. Запишіть голос, застосуйте один із 24 ретельно налаштованих ефектів (або складіть свій ланцюг) і одразу поділіться результатом у месенджерах. Запис, обробка та кодування файлу — усе на пристрої. Без реклами, облікового запису, дозволу на Інтернет та стеження.

Особливість: **ланцюги ефектів, що складаються в стек**. Pitch, реверб, ехо, дисторшн, бітхрашер та інше комбінуйте перетягуванням і зберігайте як свій голос. Більшість додатків ховають це за підпискою. EchoForge залишає безкоштовним.

## Основні можливості

### 24 пресети голосу
- **Гелій**, **Бурундук**, **Робот**, **Демон**, **Привид**, **Монстр**, **Чужий**
- **Глибокий голос**, **Велетень**, **Печера**, **Стадіон**, **Ехо**
- **Lo-Fi**, **Вініл**, **Шепіт**
- **Ванна**, **Bass Boost**, **Flanger**, **Мегафон**, **Телефон**, **Аркада**, **DJ**
- **Custom** для ваших творінь

### Налаштовувані ланцюги
- **Стек** — pitch, реверб, ехо, дисторшн, бітхрашер, EQ
- **Сортування** перетягуванням
- **Точне налаштування** повзунками
- **Скасувати / повторити**
- **Збереження** як власний пресет

### Поділитися
- **WhatsApp**, **Telegram**, **Discord**, **Messenger**, **E-mail**, **Файли**

EchoForge експортує справжні аудіофайли з правильним форматом і метаданими.

### Конфіденційність
- **100% офлайн**
- **Без реклами**
- **Без стеження**
- **Без облікового запису**
- **Аудіо лишається на пристрої**

## Сценарії

| Ситуація | Що робить EchoForge |
|----------|----------------------|
| Кумедне голосове у WhatsApp | Запис → Бурундук → поділитися |
| Хелловінське повідомлення | Запис → Демон + Печера → поділитися |
| Голосове в Discord | Запис → Робот + Bitcrush → Discord |
| ASMR-кліп | Запис → Шепіт → зберегти |
| Жарт у Telegram | Запис → Гелій → голосове |
| Тест аудіокниги | Запис → Глибокий голос → зберегти |
| Озвучка для TikTok | Запис → власний ланцюг → зберегти |
| Заставка подкасту | Запис → Мегафон / DJ → зберегти |
| Сцена «по телефону» | Запис → Телефон → поділитися |
| Голос персонажа | Запис → Монстр + Distortion → зберегти |
