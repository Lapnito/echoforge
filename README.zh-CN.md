## EchoForge 是什么？

**EchoForge** 是一款适用于 Android 与 iPhone 的完全离线变声器。录下声音，套用 24 种精心调校的效果之一（或自己堆叠效果链），将结果直接分享到通讯应用。录音、处理、音频文件编码全部在本机完成。无广告、无账号、无网络权限、无追踪。

独特之处：**可堆叠的效果链**。变调、混响、回声、失真、降采样等可以拖放组合并保存为你自己的声音。多数应用把这种自定义藏在订阅后面，EchoForge 直接免费。
